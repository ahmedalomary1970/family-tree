دوائر + روابط
أجيال + حلقات
أسماء داخل الدوائر (مع التفاف)
☁️ 2. حفظ سحابي (Supabase)
يتم حفظ:
people + links + rings + zoom + كل الإعدادات
🔗 3. نظام مشاركة
إنشاء share_token
حفظه في جدول tree_shares

🌍 4. رابط عرض (Read Only)
صفحة:
/public-tree?token=xxxx
تعرض:
نفس الشجرة
بدون تعديل
بنفس التصميم تقريبًا
🧪 5. تشغيل على الشبكة
بدل localhost

192.168.x.x
